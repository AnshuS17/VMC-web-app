# Smart Vadodara Municipality Corporation Portal

A responsive web-based portal built using **HTML, CSS, and JavaScript** to provide citizens with easy access to municipal services, complaints, notices, and information related to Vadodara Municipality.

Live site: https://anshus17.github.io/VMC-web-app/

## Project Overview

The Smart Vadodara Municipality Corporation Portal is designed to make municipal services more accessible and user-friendly. Citizens can view important information, access services, submit complaints, and stay updated with civic service updates through a clean and simple interface.

## Features

- Responsive homepage
- Citizen service sections
- Complaint registration interface
- Complaint status tracking
- Admin case management interface
- Service categories for garbage, electricity, water logging, potholes, drainage, streetlights, and other civic issues
- Location and ward details for complaints
- Clean and user-friendly UI
- Works on desktop, tablet, and mobile screens
- Static demo support on GitHub Pages

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
